# Scriberr v2.0.0 Devnotes

This folder contains the planning notes, architecture rules, sprint trackers, and feature specs for the Scriberr v2.0.0 workstream. The structure is intentionally shallow so notes are easy to scan and move as the release evolves.

## Structure

- `rules/`: Architecture, frontend, backend, design-system, and brand rules.
- `specs/`: Durable API and engine worker specifications.
- `sprint-plans/`: Planned implementation sequences by feature or workstream.
- `sprint-trackers/`: Running sprint completion trackers.
- `status-updates/`: Per-sprint notes, inventories, smoke results, and implementation updates.
