# Features

Scriberr provides a comprehensive suite of tools for audio transcription and management.

*   **State-of-the-art Models:** Run advanced local models like NVIDIA Parakeet/Canary or OpenAIâ€™s Whisper for high-accuracy results.
*   **Speaker Diarization:** Automatically identifies and labels distinct speakers throughout the conversation.
*   **Hardware Acceleration:** Utilizes NVIDIA GPUs to significantly speed up offline processing times.
*   **AI Summarization:** Connects to Ollama or OpenAI to generate concise summaries of your recordings.
*   **Chat with Transcript:** Interactive chat interface allows you to ask questions or extract insights from your audio using LLMs.
*   **Synchronized Playback:** Follow along with the audio using karaoke-style word highlighting and auto-scrolling.
*   **Click-to-Seek:** Click on any word in the text to immediately jump the audio playback to that exact moment.
*   **YouTube Transcription:** Paste a YouTube link to download and transcribe the audio directly.
*   **Multi-lingual & Translation:** Transcribe audio in dozens of languages or translate them into English during the process.
*   **Transcription Profiles:** Save your preferred model settings and parameters as reusable profiles for consistent results.
*   **PWA Support:** Install Scriberr as a native-feeling app on your desktop or mobile device.
*   **Mobile-Friendly UI:** Fully responsive design with mobile-specific interactions like swipe gestures.
*   **Audio Recorder:** Record voice notes or meetings directly within the app browser.
*   **File Watcher:** A CLI tool that monitors a folder on your computer and automatically uploads new files to Scriberr.
*   **Extensive API & Webhooks:** Full REST API support and webhooks for integrating with tools like n8n or custom scripts.
*   **Diverse Inputs:** Drag-and-drop files, upload video for audio extraction, or import multi-track `.aup` Audacity projects.
*   **Flexible Exports:** Download your data in multiple formats, including JSON, SRT subtitles, or plain text.
*   **OpenAI API Integration:** Option to offload transcription duties to the OpenAI cloud API instead of running locally.
*   **Note Taking:** Jot down time-stamped notes alongside your transcript as you listen.
*   **Bulk Actions:** Manage, delete, or re-process multiple files at once for easier library management.
